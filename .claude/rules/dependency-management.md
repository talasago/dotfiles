# 依存関係の管理

- 直接依存するパッケージのバージョンは、可能な限り完全固定する。
- 間接依存はロックファイルで再現可能な状態を維持する。
- 依存関係を追加・更新した場合は、利用可能なセキュリティ監査コマンドを実行する。
- 監査で問題が検出された場合は、脆弱性の影響・修正可能性・対応方針を確認する。
- パッケージマネージャー固有のコマンドや手順は、各ルールファイルに従う。
