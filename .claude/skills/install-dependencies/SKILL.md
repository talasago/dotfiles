# install-dependencies

既存の依存関係をインストールするための手順。依存パッケージの追加・更新は、このスキルでは行わない。

## 手順

1. 既存のlockfileを使った、再現性を優先したインストールコマンドを選ぶ。
   - pnpm: `pnpm install --frozen-lockfile --reporter=silent`
2. プロジェクトのルートディレクトリでコマンドを実行する。
3. lockfileや設定ファイルに意図しない変更がないか確認する。

## 禁止事項

- `--no-frozen-lockfile`など、依存バージョンを解決し直すオプションを通常のインストールで使用しない。
- 依存パッケージの追加・更新を、インストール作業として黙って行わない。
- プロジェクトに存在しないパッケージマネージャーのコマンドを推測して実行しない。
